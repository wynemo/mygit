这个项目最终的目标是 功能与idea这种ide里面的插件： git4idea https://github.com/JetBrains/intellij-community/tree/master/plugins/git4idea 一致
用pyqt写

第一个界面，打开文件夹，然后调用git信息，获取历史信息，形成一个列表，如果有不同的本地分支，也要展示出来

我们先来优化下界面，弄成一个列表，日志如果有多行取第一行，然后是作者，日期

然后我们需要在点击这个列表的元素的时候，列出哪些文件变化了，展示在右边的栏目

我们需要一个查看文件变化的地方，放在文件变化/提交历史 与 当前分支 这个之间，点击文件变化的文件时，就展示出来，
注意，如果是merge的情况，是三个文本框，一般是左右比较两个框
